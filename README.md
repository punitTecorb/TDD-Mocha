# TDD-Mocha using nodejs with TypeScript 

## Introduction
   Test cases using mocha and chai library.We have create demo test case.
    
## Setup Steps:
`npm install`
### local server
`nodemon server.ts`
### install mocha
`npm i @types/mocha`
### install chai
`npm i @types/chai`
### install request
`npm i @types/request`
### use this script
`"test": "mocha -r ts-node/register src/test/**/*_spec.ts"`
### testing command
`npm test`
